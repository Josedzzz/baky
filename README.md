# baky

baky is a terminal-based backup utility designed for sysadmins to quickly manage and automate backups of local files and directories to a NAS. Built with Go and the Bubble Tea TUI framework, it provides a fast, keyboard-driven interface for server maintenance.

## Tech Stak

- Language: Go (https://go.dev/) (1.24+)
- TUI Framework: Bubble Tea (https://github.com/charmbracelet/bubbletea)
- UI Components: Bubbles (https://github.com/charmbracelet/bubbles)
- Styling: Lip Gloss (https://github.com/charmbracelet/lipgloss)
